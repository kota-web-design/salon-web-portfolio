オンラインスクールLP｜成約につながる構成設計のLP制作事例

※画像クリックで実際のサイトをご覧いただけます

概要

オンラインスクールの受講者獲得を目的としたLPです。
学習によって得られる未来を明確にし、受講への不安を解消する構成で制作しました。

想定ターゲット：スキル習得・キャリアアップを目指す20〜30代
目的：受講申し込み・無料相談の獲得

担当範囲

  • 担当範囲コーディング
  • 対応内容HTML / CSS / JavaScript / レスポンシブ対応
  • 想定支給物Figmaなどのデザインカンプ

制作意図

オンラインスクールは「本当に成果が出るのか」という不安が大きいため、
未来の変化を具体的に提示し、信頼につなげる設計を意識しています。

シンプルで視認性の高いデザインにより、情報をストレスなく伝えることを重視しました。

設計のポイント

本LPは「未来提示 → 信頼 → 行動」の流れで構成しています。

ファーストビューで理想の未来を提示した後、
サービス内容 → カリキュラム → 実績 → 受講者の声 → CTAの順に配置し、
ユーザーの不安を段階的に解消する設計にしています。

特に「成果イメージ」を具体化することで、
受講後の自分を想像しやすくしています。

想定される効果

本LPは、未来提示と信頼構築により、
「興味 → 理解 → 信頼 → 行動」へと自然に誘導する構成になっています。

その結果、受講申し込み率の向上や、
無料相談・問い合わせにつながるコンバージョン改善が期待できます。

工夫したポイント

  • 未来の変化を明確に伝えるファーストビュー設計
  • カリキュラムと実績で信頼を構築
  • 受講者の声を配置し安心感を強化
  • 情報を整理したシンプルで見やすいUI
  • スマートフォンでも理解しやすいレスポンシブ設計

他と違うポイント

見た目のデザインだけでなく、
「受講後にどう変化するか」を具体的に伝える設計を行っている点が強みです。

学習サービスにおいて重要な「未来提示」と「信頼構築」を、
スマートフォンでも伝わりやすい形で実装しています。

制作期間

7日
（HTML/CSSコーディング・レスポンシブ対応含む）

  • JavaScript（UI制御・アニメーション）
  • レスポンシブ対応（SP / PC）
