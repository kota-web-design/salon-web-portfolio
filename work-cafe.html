カフェLP｜来店したくなる世界観設計のLP制作事例

※画像クリックで実際のサイトをご覧いただけます

概要

夜の時間帯をメインターゲットにしたカフェを想定したLPです。
落ち着いた雰囲気と居心地の良さを伝えながら、来店につながる導線設計を意識して制作しました。

想定ターゲット：20〜30代の仕事帰り層・夜カフェ利用者
目的：来店動機の創出・来店率向上

担当範囲

  • 担当範囲コーディング
  • 対応内容HTML / CSS / JavaScript / レスポンシブ対応
  • 想定支給物Figmaなどのデザインカンプ

制作意図

「仕事終わりに立ち寄りたくなる空間」をコンセプトに設計しています。
照明や写真のトーン、余白設計を活かし、リラックスできる世界観を表現しました。

設計のポイント

本LPは「来店したくなる心理導線」を重視して構成しています。

ファーストビューで雰囲気を伝えた後、
コンセプト → メニュー → 店内の様子 → 利用シーン → CTA の流れで設計することで、
自然に来店イメージができる構成にしています。

また、視覚情報を中心に構成することで、
直感的に魅力が伝わる設計にしています。

想定される効果

本LPは、視覚的な魅力とストーリー設計により、
「興味 → 共感 → 来店イメージ → 行動」へと自然に誘導する構成になっています。

その結果、来店意欲の向上や、
実際の来店行動につながるコンバージョン改善が期待できます。

工夫したポイント

  • 夜カフェの雰囲気を写真と配色で直感的に伝えるデザイン
  • 余白とタイポグラフィを活かした落ち着きのあるレイアウト
  • 「興味 → 共感 → 来店イメージ → 行動」の流れを意識した構成
  • メニューや空間を視覚的に見せることで来店動機を強化
  • スマートフォンでもストレスなく閲覧できるレスポンシブ設計

他と違うポイント

見た目のデザインだけでなく、
「どうすれば来店したくなるか」という視点で設計している点が強みです。

ユーザー導線を意識した構成設計と、
それを正確に再現するコーディングを一貫して行っています。

制作期間

約5日
（HTML/CSSコーディング・レスポンシブ対応含む）
※内容やボリュームにより変動します

使用技術

  • HTML / CSS（セマンティック設計）
  • JavaScript（UI制御・アニメーション）
  • レスポンシブ対応（SP / PC）
