タロット占いLP｜共感と信頼で行動につなげるLP制作事例

※画像クリックで実際のサイトをご覧いただけます

概要

タロット占いサービスを想定したLPです。
ユーザーの不安や悩みに寄り添いながら、安心して相談できる導線設計を意識して制作しました。

想定ターゲット：悩みや不安を抱える20〜40代女性
目的：相談予約・問い合わせ獲得

担当範囲

  • 担当範囲コーディング
  • 対応内容HTML / CSS / JavaScript / レスポンシブ対応
  • 想定支給物Figmaなどのデザインカンプ

制作意図

占いサービスに対する「怪しい」「不安」といった心理的ハードルを下げ、
共感と安心感を与えることで行動につなげることを目的としています。

ダークトーンをベースにしつつ、余白や装飾で高級感と神秘性を表現しました。

設計のポイント

本LPは「共感 → 信頼 → 行動」の流れを意識して構成しています。

ファーストビューで世界観を伝えた後、
悩みの共感 → サービス内容 → 実績・信頼 → CTAの順に配置し、
ユーザーの心理に沿った自然な導線を設計しています。

特に「悩み共感セクション」を強く設計することで、
ユーザー自身の状況とサービスを結びつける構成にしています。

想定される効果

本LPは、共感と信頼の構築により、
「悩み → 共感 → 安心 → 行動」へと自然に誘導する構成になっています。

その結果、相談予約率の向上や、
問い合わせにつながるコンバージョン改善が期待できます。

工夫したポイント

  • ダークトーンと装飾を活かした神秘的な世界観デザイン
  • 悩み共感から自然にサービス理解へつなげる構成設計
  • 信頼要素（実績・プロフィール）を適切に配置
  • CTAを適切なタイミングで配置し行動を促進
  • スマートフォンでも没入感を維持したレスポンシブ設計

他と違うポイント

見た目のデザインだけでなく、
「どうすれば相談につながるか」という視点で設計している点が強みです。

ユーザーの不安に寄り添う構成を、
世界観を崩さないレスポンシブ表示で実装しています。

制作期間

8日
（HTML/CSSコーディング・レスポンシブ対応含む）

使用技術

  • HTML / CSS（セマンティック設計）
  • JavaScript（UI制御・アニメーション）
  • レスポンシブ対応（SP / PC）
